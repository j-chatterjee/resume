How It Works

  1. Edit your portfolio content in any panel on the left.
  2. Click Save Draft to store changes locally in your browser.
  3. Configure your GitHub token and repo settings above.
  4. Click Publish to GitHub (sidebar) to commit changes directly to your repo.
  5. GitHub Pages rebuilds automatically – your live site updates within ~60 seconds.
